# app > clientNavbar

*As of v8.1.*

Navbar definitions.

Example:

```json
{
    "itemDefs": {
        "quickCreate": {
            "view": "views/site/navbar/quick-create",
            "class": "dropdown hidden-xs quick-create-container"
        }
    },
    "itemList": [
        "quickCreate"
    ]
}
```

## itemDefs

*Object*

Definitions.

### view

*string*

A frontend view.

### class

*string*

A CSS class or multiple classes for the LI tag.
